Get Started

Fluentd is a JSON-based, lightweight log collector daemon. It receives records from your applications or other software and writes them to RDBMS, NoSQL (e.g. MongoDB, Redis), flat files, or even other instances of Fluentd.

Install Fluentd
RPM package (.rpm) Debian package (.deb) gem install fluentd
