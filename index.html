Get Started

Fluentd is a JSON-based, lightweight logging daemon for real-time analysis. It receives events from your application or other logging software and writes them to RDBMS, NoSQL, flat files, or even other instances of Fluentd.

Install Fluentd
Gem install fluentd Debian package (.deb) RPM package (.rpm)
